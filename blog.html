Assignment 3 11/8/25

A challenge I faced when completing this assignment was the filter section. The first issue I ran into was that my code wasn't being applied to the posts. After spending a while trying to find the problem, I realized that I had been using "class" to reference something, but I should have been using "name". The class I had been using didn't exist which is why none of my code was being applied. This assignment taught me a lot about the basics of javascript. I learned how to create and implement variables, functions, and loops. I learned how to access different elements throughout the document using querySelect and getElementBy. The lecture videos were helpful for this assignment. If I was stuck, I would go back and watch the videos.
